# A helper to add basic tooltip inside a gt table

This is a lightweight helper to add tooltip, typically to be used within
[`gt::cols_label()`](https://gt.rstudio.com/reference/cols_label.html).

## Usage

``` r
with_tooltip(label, tooltip)
```

## Arguments

- label:

  The label for the item with a tooltip

- tooltip:

  The text based tooltip for the item

## Value

HTML text
